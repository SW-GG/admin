# 🧇 Waffle Order Management System

## 📝 Project Description
와플 주문을 관리하는 웹 애플리케이션입니다. 주문 요청, 주문 대기, 주문 완료 상태를 관리하고 실시간으로 주문 현황을 모니터링할 수 있습니다.

## ✨ Features

### 주문 관리
- 주문 상태별 필터링 (주문요청/주문대기중/주문완료)
- 날짜별 주문 조회
- 페이지네이션
- 주문 상태 변경
- 실시간 주문 현황 모니터링

### 주문 기능
- 기본 와플 / 초코 와플 수량 선택
- 전화번호 입력
- 요청사항 메모
- 총 금액 자동 계산

### 기타 기능
- 토스트 알림

## 🛠 Tech Stack

### Frontend
- **Next.js** (v14.2.13)
- **React** (v18)
- **JavaScript**
- **Tailwind CSS**

### Backend
- **Supabase** (Backend as a Service)
- **PostgreSQL** (Database)

### API
- **REST API**
- **Next.js API Routes**
